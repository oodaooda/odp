# OpenClaw Handoff Checklist

## 1. Specs
- PRD/SRD/PDR/ICD/DDR/V&V/Decisions present
- Runbook, SLOs, Config/Secrets included

## 2. Milestone 1
- Scope defined
- Gates defined
- Evidence requirements defined

## 3. Data & Memory
- Postgres schema documented
- pgvector usage documented
- Compaction policy documented

## 4. UI
- Dashboard + Task Detail + Gate Evidence + Chat mockups present
- UX confirmation flows specified

## 5. Ops
- Runbook ready
- Backup/restore steps defined
