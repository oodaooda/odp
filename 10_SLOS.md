- Orchestrator availability: 99.5% (local dev target)
- UI availability: 99.5%

- Max concurrent tasks per project: 5 (initial)
- Task completion time tracked per phase

## 4. Quality
- Gate pass rate tracked per phase
- Regression escapes: 0 tolerated
